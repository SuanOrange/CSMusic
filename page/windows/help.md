# 常见问题
此帮助文档基于Countless Simple Music V2.0.9所写，无法保证其可用性在100%

## 为什么有些歌曲一播放软件就闪退
简繁音乐Windows客户端采用的编码为UTF-8，而UTF-8编码对韩语、维吾尔语、蒙古语等多种语音并不支持，这就导致在播放一些其他语种的歌曲包括但不限于韩语的歌曲时，简繁音乐无法正常对歌名、歌词进行解码，所以会导致软件闪退。
